# 🛒 ShopFlow Automation

> A Selenium + Cucumber automation framework for testing real-world e-commerce workflows.

As an SDET, I wanted to build a project that combines UI and API automation in one place while following the same practices used in enterprise QA teams. This project focuses on automating common customer journeys and validating backend APIs.

---

## 🚀 Tech Stack

* ☕ Java
* 🌐 Selenium WebDriver
* 🥒 Cucumber BDD
* 🧪 TestNG
* 🔌 REST Assured
* 📦 Maven
* ⚙️ GitHub Actions
* 📊 Extent Reports

---

## 🎯 Automated Scenarios

### 🧑 Authentication

* User Registration
* Login & Logout
* Invalid Login Validation

### 🛍️ Product Management

* Product Search
* Product Details Validation
* Add/Remove Products from Cart

### 🛒 Checkout Flow

* Cart Validation
* Checkout Process
* Order Confirmation

### 🔗 API Testing

* Products API
* Brands API
* Search API
* User API Validation

---

## ✨ Framework Features

* Page Object Model (POM)
* Cucumber BDD
* Data-Driven Testing
* Parallel Execution
* Screenshot Capture on Failure
* Extent Reporting
* CI/CD Integration
